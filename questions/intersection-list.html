Intersection of Lists

Description

You are given two integer arrays list1 and list2. Find the intersection node (first common element by reference/value) of the two lists if it exists.

Return the intersecting value, or -1 if there is no intersection.

Example 1:

Sample Input

n = 5, m = 4

list1 = [1, 2, 3, 4, 5]

list2 = [9, 8, 3, 4]

Sample Output

3

Example 2:

Sample Input

n = 4, m = 4

list1 = [7, 6, 1, 9]

list2 = [3, 2, 1, 9]

Sample Output

1
