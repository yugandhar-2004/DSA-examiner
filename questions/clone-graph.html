Clone Graph

Description

Given a graph represented as an adjacency list for each node from 1 to n, clone the graph and return the cloned list representation.

The graph is represented as an array of lists, where each list contains the neighbors of the node.

Example 1:

Sample Input

n = 4

adjList = [[2, 4], [1, 3], [2, 4], [1, 3]]

Sample Output

[[2, 4], [1, 3], [2, 4], [1, 3]]

Example 2:

Sample Input

n = 1

adjList = [[]]

Sample Output

[[]]
