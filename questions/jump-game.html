Jump Game

Description

Given an integer n and an array of n non-negative integers, where each element represents your maximum jump length at that position, determine if you can reach the last index from the first index.

Return true or false.

Example 1:

Sample Input

n = 5

nums = [2, 3, 1, 1, 4]

Sample Output

true

Example 2:

Sample Input

n = 5

nums = [3, 2, 1, 0, 4]

Sample Output

false
