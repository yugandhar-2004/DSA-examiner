Reverse First K Elements of Queue

Description

Given an integer n, an array of n integers representing a queue, and an integer k, reverse the first k elements of the queue.

Print the updated queue as an array.

Example 1:

Sample Input

n = 5

arr = [1, 2, 3, 4, 5]

k = 3

Sample Output

[3, 2, 1, 4, 5]

Example 2:

Sample Input

n = 4

arr = [10, 20, 30, 40]

k = 2

Sample Output

[20, 10, 30, 40]
