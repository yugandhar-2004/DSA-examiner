Reverse a Queue

Description

Given an integer n, and a queue of n integers, reverse the queue and print the result.

Example 1:

Sample Input

n = 5

queue = [1, 2, 3, 4, 5]

Sample Output

[5, 4, 3, 2, 1]

Example 2:

Sample Input

n = 4

queue = [10, 20, 30, 40]

Sample Output

[40, 30, 20, 10]
