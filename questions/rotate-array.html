Rotate Array

Description

Given an integer n, an array of n integers nums, and an integer k, rotate the array to the right by k steps.

Do it in-place with O(1) extra space.

Example 1:

Sample Input

n = 7

nums = [1, 2, 3, 4, 5, 6, 7]

k = 3

Sample Output

[5, 6, 7, 1, 2, 3, 4]

Example 2:

Sample Input

n = 5

nums = [10, 20, 30, 40, 50]

k = 2

Sample Output

[40, 50, 10, 20, 30]
