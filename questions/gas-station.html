Gas Station

Description

Given two integer arrays gas and cost each of length n. You have n gas stations arranged in a circle. You start at one station with an empty tank. At each station, you can fill gas[i] units and spend cost[i] units to go to the next station. Return the starting station index if you can travel around the circuit once, otherwise return -1.

Example 1:

Sample Input

n = 5

gas = [1, 2, 3, 4, 5]

cost = [3, 4, 5, 1, 2]

Sample Output

3

Example 2:

Sample Input

n = 3

gas = [2, 3, 4]

cost = [3, 4, 3]

Sample Output

-1
