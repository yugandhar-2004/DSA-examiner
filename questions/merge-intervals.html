Merge Intervals

Description

Given an integer n and an array of n pairs representing intervals, merge all overlapping intervals and return them in sorted order.

Example 1:

Sample Input

n=4

intervals = [[1,3], [2,6], [8,10], [15,18]]

Sample Output

[[1,6],[8,10],[15,18]]

Example 2:

Sample Input

n=2

intervals = [[1,4], [4,5]]

Sample Output

[[1,5]]
