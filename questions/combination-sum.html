Combination Sum

Description

Given an integer n, an array of n integers, and a target value, return all unique combinations where the numbers sum up to the target.

Each number can be used unlimited times.

Example 1:

Sample Input

n = 4

nums = [2, 3, 6, 7]

target = 7

Sample Output

[[2, 2, 3], [7]]

Example 2:

Sample Input

n = 3

nums = [2, 3, 5]

target = 8

Sample Output

[[2, 2, 2, 2], [2, 3, 3], [3, 5]]
