Generate Binary Numbers from 1 to N

Description

Given an integer n, generate binary representations of numbers from 1 to n.

Print the result as an array of binary strings.

Example 1:

Sample Input

n = 5

Sample Output

["1", "10", "11", "100", "101"]

Example 2:

Sample Input

n = 3

Sample Output

["1", "10", "11"]
