Permutations II

Description

Given an integer n, and an array of n integers (which may contain duplicates), return all unique permutations.

Example 1:

Sample Input

n = 3

nums = [1, 1, 2]

Sample Output

[[1, 1, 2], [1, 2, 1], [2, 1, 1]]

Example 2:

Sample Input

n = 3

nums = [1, 2, 2]

Sample Output

[[1, 2, 2], [2, 1, 2], [2, 2, 1]]
