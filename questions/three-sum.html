Three sum

Description

Given an integer n, an array of n integers nums, and an integer target, return indices of three numbers such that they add up to the target.

You may assume that each input has exactly one solution, and you may not use the same element more than once.

Return the answer in the form of a list of three integers (indices).

Example 1:

Sample Input

n=6

nums = [5, 7, 1, 2, 8, 4]

target = 10

Sample Output

[0, 2, 5]

Example 2:

Sample Input

n=5

nums = [5, 12, 1, 3, 6]

target = 9

Sample Output

[0, 2, 3]
