Contains Duplicate

Description

Given an integer n and an array of n integers nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

Example 1:

Sample Input

n=5

nums = [1, 2, 3, 4, 1]

Sample Output

true

Example 2:

Sample Input

n=4

nums = [1, 2, 3, 4]

Sample Output

false
