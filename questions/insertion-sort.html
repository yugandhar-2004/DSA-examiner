Insertion sort

Description

Given an integer n, and an array of n integers, sort the array using the Insertion sort algorithm.

Print the sorted array.

Example 1:

Sample Input

n = 6

nums = [64, 25, 12, 22, 11, 90]

Sample Output

[11, 12, 22, 25, 64, 90]

Example 2:

Sample Input

n = 5

nums = [5, 3, 4, 1, 2]

Sample Output

[1, 2, 3, 4, 5]
