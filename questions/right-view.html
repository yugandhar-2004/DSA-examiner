Binary Tree Right Side View

Description

You are given an integer n and an array of n integers representing a binary tree in level-order form (with -1 as null).

Return the values of the nodes that are visible from the right side, in order from top to bottom.

Example 1:

Sample Input

n = 7

tree = [1, 2, 3, -1, 5, -1, 4]

Sample Output

[1, 3, 4]

Example 2:

Sample Input

n = 1

tree = [7]

Sample Output

[7]
