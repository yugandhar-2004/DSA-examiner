Rotate List

Description

Given an integer n, an array of n integers representing a linked list, and an integer k, rotate the list to the right by k places.

Return the rotated list as an array.

Example 1:

Sample Input

n = 5, k = 2

list = [1, 2, 3, 4, 5]

Sample Output

[4, 5, 1, 2, 3]

Example 2:

Sample Input

n = 3, k = 1

list = [0, 1, 2]

Sample Output

[2, 0, 1]
