Longest Increasing Subsequence

Description

Given an integer n, and an array of n integers, find the length of the longest strictly increasing subsequence.

Example 1:

Sample Input

n = 6

nums = [10, 9, 2, 5, 3, 7]

Sample Output

3

Example 2:

Sample Input

n = 6

nums = [0, 1, 0, 3, 2, 3]

Sample Output

4
