Minimum Cost to Connect All Points

Description

You are given an array of points where points[i] = [xi, yi] represents a point on the 2D plane. Return the minimum total cost to connect all points. The cost to connect two points is the Manhattan distance between them.

Example 1:

Sample Input

n = 3

points = [[0, 0], [2, 2], [3, 10]]

Sample Output

15

Example 2:

Sample Input

n = 4

points = [[0, 0], [1, 1], [1, 0], [-1, 1]]

Sample Output

4
