Buy and Sell Stock

Description

Given an integer n and an array of n integers prices, where prices[i] is the price of a given stock on the ith day, you must find the maximum profit you can achieve from a single buy and a single sell.

You must buy before you sell.

If no profit can be achieved, return 0.

Example 1:

Sample Input

n=6

prices = [7, 1, 5, 3, 6, 4]

Sample Output

5

Example 2:

Sample Input

n=5

prices = [7, 6, 4, 3, 1]

Sample Output

0
