Implement Queue

Description

Implement a basic queue using an array. You are given an integer q, the number of operations followed by q operations. The operations can be "push x" or "pop". Return the final state of the queue.

Example 1:

Sample Input

q = 5

ops = ["push 1", "push 2", "pop", "push 3", "push 4"]

Sample Output

[2, 3, 4]

Example 2:

Sample Input

q = 4

ops = ["push 10", "push 20", "pop", "pop"]

Sample Output

[]
