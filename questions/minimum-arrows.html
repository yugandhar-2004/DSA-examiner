Minimum Number of Arrows

Description

You are given an array of balloons represented by intervals [start, end]. One arrow can burst all balloons that intersect with it. Return the minimum number of arrows required to burst all balloons.

Example 1:

Sample Input

n = 4

balloons = [[10, 16], [2, 8], [1, 6], [7, 12]]

Sample Output

2

Example 2:

Sample Input

n = 4

balloons = [[1, 2], [3, 4], [5, 6], [7, 8]]

Sample Output

4
