Sort Colors

Description

Given an integer n, and an array of n integers where each element is 0, 1, or 2, sort the array in-place so that all 0s come first, followed by all 1s, then all 2s.

Return the sorted array.

Example 1:

Sample Input

n = 6

nums = [2, 0, 2, 1, 1, 0]

Sample Output

[0, 0, 1, 1, 2, 2]

Example 2:

Sample Input

n = 4

nums = [1, 0, 2, 1]

Sample Output

[0, 1, 1, 2]
