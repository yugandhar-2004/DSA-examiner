Merge Two Sorted Lists

Description

You are given two sorted arrays. Merge them into one sorted list.

Return the merged sorted array.

Example 1:

Sample Input

n = 3, m = 3

list1 = [1, 3, 5]

list2 = [2, 4, 6]

Sample Output

[1, 2, 3, 4, 5, 6]

Example 2:

Sample Input

n = 2, m = 4

list1 = [1, 7]

list2 = [2, 3, 5, 6]

Sample Output

[1, 2, 3, 5, 6, 7]
