Delete Node in a Linked List

Description

You are given an integer n, followed by a list of n integers representing a singly linked list.

You are also given an index of a node (not the tail). Delete that node from the list.

Return the modified linked list as an array.

Example 1:

Sample Input

n = 5

list = [4, 5, 1, 9, 7]

deleteIndex = 1

Sample Output

[4, 1, 9, 7]

Example 2:

Sample Input

n = 4

list = [1, 2, 3, 4]

deleteIndex = 2

Sample Output

[1, 2, 4]
