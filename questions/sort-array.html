Sort Array

Description

Given an integer n, and an array of n integers, return the array sorted in ascending order.

Return the sorted array.

Example 1:

Sample Input

n = 5

nums = [5, 1, 4, 2, 3]

Sample Output

[1, 2, 3, 4, 5]

Example 2:

Sample Input

n = 4

nums = [10, -3, 2, 8]

Sample Output

[-3, 2, 8, 10]
