Remove K Digits

Description

Given an integer n, a string representing a non-negative integer of length n, and an integer k, remove k digits from the number so that the new number is the smallest possible.

Return the resulting number as a string.

Example 1:

Sample Input

n = 4, num = "1432", k = 2

Sample Output

12

Example 2:

Sample Input

n = 5, num = "10200", k = 1

Sample Output

200
