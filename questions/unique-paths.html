Unique Paths

Description

A robot is located at the top-left corner of an m x n grid. The robot can only move either down or right at any point in time. Find how many possible unique paths are there to reach the bottom-right corner.

Example 1:

Sample Input

m = 3, n = 7

Sample Output

28

Example 2:

Sample Input

m = 3, n = 2

Sample Output

3
