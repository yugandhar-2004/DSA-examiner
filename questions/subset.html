Subsets

Description

Given an integer n, and an array of n unique integers, return all possible subsets (the power set).

Return the answer as a list of lists.

Example 1:

Sample Input

n = 3

nums = [1, 2, 3]

Sample Output

[[ ], [1], [2], [3], [1, 2], [1, 3], [2, 3], [1, 2, 3]]

Example 2:

Sample Input

n = 2

nums = [0, 1]

Sample Output

[[ ], [0], [1], [0, 1]]
