Remove N-th Node from End of List

Description

Given an integer n, and a singly linked list, remove the n-th node from the end of the list and return the modified list.

Example 1:

Sample Input

size = 5, remove = 2

list = [1, 2, 3, 4, 5]

Sample Output

[1, 2, 3, 5]

Example 2:

Sample Input

size = 1, remove = 1

list = [10]

Sample Output

[]
