Valid Parentheses

Description

Given an integer n and a string with n characters consisting only of '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

An input string is valid if:

  • Open brackets are closed by the same type of brackets.
  • Open brackets are closed in the correct order.

Return true or false.

Example 1:

Sample Input

n = 6, s = "()[]{}"

Sample Output

true

Example 2:

Sample Input

n = 5, s = "([)]"

Sample Output

false
