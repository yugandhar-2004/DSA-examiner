Non-overlapping Intervals

Description

Given an array of n intervals where intervals[i] = [starti, endi], return the minimum number of intervals you need to remove to make the rest of the intervals non-overlapping.

Example 1:

Sample Input

n = 4

intervals = [[1,2], [2,3], [3,4], [1,3]]

Sample Output

1

Example 2:

Sample Input

n = 3

intervals = [[1,2], [1,2], [1,2]]

Sample Output

2
