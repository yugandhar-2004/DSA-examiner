Palindrome List

Description

Given an integer n, and a list of n integers, determine if the list is a palindrome. A list is a palindrome if it reads the same forward and backward.

Return "true" if it is a palindrome, otherwise return "false".

Example 1:

Sample Input

n = 5

nums = [1, 2, 3, 2, 1]

Sample Output

true

Example 2:

Sample Input

n = 4

nums = [1, 2, 2, 3]

Sample Output

false
