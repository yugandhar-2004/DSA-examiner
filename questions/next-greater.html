Next Greater Element

Description

Given an integer n, and an array of n integers, return an array where for each element, the next greater element is shown. If no such element exists, use -1.

Return the result as an array.

Example 1:

Sample Input

n = 4

nums = [2, 1, 2, 4]

Sample Output

[4, 2, 4, -1]

Example 2:

Sample Input

n = 5

nums = [1, 3, 2, 4, 1]

Sample Output

[3, 4, 4, -1, -1]
