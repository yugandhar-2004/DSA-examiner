Find Center of Star Graph

Description

Given n edges in a graph,A star graph is a graph where there is one center node connected to all other nodes. Given an array of edges representing an undirected star graph, return the center of the graph.

Example 1:

Sample Input

n=3

edges = [[1, 2], [2, 3], [4, 2]]

Sample Output

2

Example 2:

Sample Input

n=4

edges = [[5, 1], [2, 5], [5, 3], [4, 5]]

Sample Output

5
