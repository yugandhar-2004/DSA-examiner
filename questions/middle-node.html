Middle of Linked List

Description

Given an integer n and an array of n integers representing a singly linked list, return the value of the middle node.

If there are two middle nodes, return the second middle value.

Example 1:

Sample Input

n = 5

list = [1, 2, 3, 4, 5]

Sample Output

3

Example 2:

Sample Input

n = 6

list = [1, 2, 3, 4, 5, 6]

Sample Output

4
