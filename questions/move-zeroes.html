Move Zeroes

Description

Given an integer n and an array of n integers nums, move all 0's to the end of the array while maintaining the relative order of the non-zero elements.

You must do this in-place without making a copy of the array.

Return the modified array.

Example 1:

Sample Input

n = 5

nums = [0, 1, 0, 3, 12]

Sample Output

[1, 3, 12, 0, 0]

Example 2:

Sample Input

n = 6

nums = [0, 0, 1, 2, 0, 3]

Sample Output

[1, 2, 3, 0, 0, 0]
