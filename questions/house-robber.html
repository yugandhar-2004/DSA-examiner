House Robber

Description

You are a professional robber planning to rob houses along a street. Each house has a certain amount of money. You cannot rob two adjacent houses. Return the maximum amount you can rob tonight without alerting the police.

Example 1:

Sample Input

n = 4

nums = [1, 2, 3, 1]

Sample Output

4

Example 2:

Sample Input

n = 5

nums = [2, 7, 9, 3, 1]

Sample Output

12
